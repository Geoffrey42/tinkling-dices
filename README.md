# tinkling-dices
a little rooms booking platform for STATION F

## Installation

### Requirements
* Docker
* Docker-compose

```shell
$ git clone https://github.com/Geoffrey42/tinkling-dices.git
$ cd tinkling-dices/
```

## Usage

```shell
$ docker-compose up
```
